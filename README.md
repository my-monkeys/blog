# blog.my-monkey.fr

Static blog for My-Monkey collective. Astro 5 + MDX + Decap CMS + auto-changelog from GitHub Releases.

See `CLAUDE.md` for full project conventions and setup instructions.

## Quick start

```bash
npm install
npm run dev          # http://localhost:4321
npm run build        # → dist/
npm test
```
